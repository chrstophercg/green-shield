# 🛡 GreenShield

> **Monitorando hoje para proteger o amanhã.**

O **GreenShield** é uma plataforma open source de monitoramento ambiental desenvolvida para transformar dados públicos em informações claras e acessíveis sobre incidentes ambientais próximos a áreas protegidas.

O projeto tem como objetivo auxiliar pesquisadores, jornalistas, organizações da sociedade civil e cidadãos interessados na conservação ambiental, oferecendo uma visão integrada de dados geoespaciais por meio de mapas interativos e indicadores de risco.

---

## 🌎 Objetivo

O GreenShield busca responder a uma pergunta simples:

> **Quais áreas protegidas precisam de atenção agora e por quê?**

Em vez de apenas apresentar dados brutos, a plataforma organiza e contextualiza informações para facilitar a identificação de áreas sob maior pressão ambiental.

## 📊 Fontes de dados

O GreenShield utilizará exclusivamente dados públicos disponibilizados por órgãos oficiais e iniciativas abertas.

Entre as principais fontes previstas estão:

- INPE
- FUNAI
- ICMBio
- IBAMA
- MapBiomas
- OpenStreetMap

## ⚠️ Aviso

O GreenShield é um projeto independente de código aberto.

A plataforma organiza e apresenta dados públicos para facilitar análises ambientais, mas não substitui órgãos oficiais de fiscalização nem realiza classificações jurídicas sobre eventuais infrações ambientais.

## 💚 Sobre

O GreenShield nasceu da ideia de utilizar tecnologia e dados públicos para apoiar a proteção do patrimônio ambiental brasileiro.

Mais do que apresentar mapas, o objetivo é fornecer uma ferramenta simples, acessível e transparente para monitorar áreas ambientalmente sensíveis e incentivar o uso responsável de dados abertos.
